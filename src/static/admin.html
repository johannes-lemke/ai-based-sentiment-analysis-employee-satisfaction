Kategorien

Themen, nach denen das Feedback ausgewertet wird.

Die KI ordnet jedes Feedback einer dieser Kategorien zu und bewertet die Stimmung dafür (negativ, neutral, positiv). Wird eine Kategorie geändert, ergänzt oder entfernt, ändern sich Zuordnung und Score. Vorhandenes Feedback wird deshalb beim Speichern ab dem gewählten Stichtag neu ausgewertet.

Bereiche

Auswahlmöglichkeiten im Feedback-Formular.

Neu auswerten?

Beim Speichern wird das Feedback ab dem Stichtag neu ausgewertet. Der Server ist dabei für einige Zeit nicht erreichbar. Am besten abends ausführen.
